Tool: Use first moment method for maximum upper bound

One-Sentence Summary

Nobody can be above level \(m\) if the expected number of people above \(m\) is tiny: \(\mathbb{P}(\max > m) \le \mathbb{E}[\#\{i : X_i > m\}]\).

What Problem Does This Solve?

Upper bounds on maxima of large families of random variables — BRW particles, DGFF heights, combinatorial counts.

Input

A family \(\{X_i\}_{i \le N}\) with computable marginal tails. No independence needed — that’s the luxury of this direction.

Output

\(\mathbb{P}(\max_i X_i > m) \le N \cdot \max_i \mathbb{P}(X_i > m)\); the critical \(m^*\) is where the right side crosses \(1\): \[\text{(entropy) } \log N = -\log \mathbb{P}(X > m^*) \text{ (energy)}.\]

What It Converts

  • from: a maximum over exponentially many dependent variables
  • to: one single-variable tail estimate plus a count

Core Idea

Markov’s inequality on the count \(Z_m = \#\{i: X_i > m\}\). For BRW, compute \(\mathbb{E} Z_m\) with the many-to-one lemma (Branching random walk) and a Cramér tail; for the DGFF, with the Gaussian tail and \(\mathrm{Var} = \frac{2}{\pi}\log N\). Both give the correct leading order of the max — dependence costs nothing at first order.

Reflex calculation (DGFF, \(d=2\)): \(N^2\) variables, variance \(g\log N\), tail \(e^{-m^2/(2g\log N)}\); solve \(N^2 e^{-m^2/(2g\log N)} = 1\) ⇒ \(m^* = 2\sqrt{g}\log N\). Matches the true leading order.

When Should I Think of This?

  • Any “how high does the maximum reach?” question: run this first, always. It costs ten minutes and calibrates everything.
  • When the count is over paths/particles with exponential growth — pair with Use Cramer transform for large deviation speed.

Limits of the Tool

It cannot see corrections: the true max sits a \(\log\) below \(m^*\) (the \(\frac{3}{2\theta^*}\log n\) of BRW, the \(\frac34\sqrt{g}\log\log N\) of DGFF), because paths that get high early get killed by their descendants’ failure — capturing that needs barriers and Use multiscale decomposition. And it gives no lower bound: that’s Use second moment method for existence lower bound.

Where This Appears in My Path

First and second moment arguments were the daily bread of Jinyoung Park’s probabilistic combinatorics seminar (Spring 2026); the same reflex transplanted to BRW/DGFF is how the Bourgade reading connected back.

Current Understanding

Automatic at leading order.

Open Confusions
