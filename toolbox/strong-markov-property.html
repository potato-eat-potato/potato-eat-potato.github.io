Theorem

Theorem: Strong Markov property

Statement

For a Markov chain and a stopping time \(\tau\) with \(\tau < \infty\): conditionally on \(\mathcal{F}_\tau\) and \(\{X_\tau = x\}\), the shifted process \((X_{\tau + n})_{n \ge 0}\) is the chain started from \(x\), independent of the past.

Intuition

The walk has no memory even at random times — as long as the random time doesn’t peek into the future.

When to Use

The decomposition license. Every “split the path at the first visit to…” argument is an application:

  • Visit counts are geometric: visits to \(x\) before exiting \(D\) = geometric with success probability \(\mathbb{P}_x(\text{no return before exit})\). One line later: \(G_D(x,x) = c(x)R_{\mathrm{eff}}(x, D^c)\) — the identity joining my two reading projects (see Green function).
  • Excursion decompositions; last-exit-style formulas (with care, see below).
  • First-entrance decompositions for Green function identities (e.g. the puncture identity in Killed Green function has a strong-Markov proof alongside the Gaussian one).

What It Converts

One long path into i.i.d. blocks between regeneration times — geometry into geometric random variables.

Typical Failure Modes

The “time” must be a stopping time. Last exit times are not stopping times — that’s why last-exit decompositions (used for Equilibrium potential formulas) need a separate argument (reverse the walk), not a naive strong Markov application. This trips me up; flagging it permanently.
