Tool: Use Cramer transform for large deviation speed

One-Sentence Summary

To find how fast an exponentially unlikely event decays — or how far an exponentially large population spreads — compute \(\Lambda\), transform to \(\Lambda^*\), and solve a one-variable equation.

What Problem Does This Solve?

Finding the exponential rate of rare events for sums (\(\mathbb{P}(S_n \ge an) \approx e^{-n\Lambda^*(a)}\)), and speed selection: the front location for populations of exponentially growing size.

Input

The step distribution, via its log-MGF \(\Lambda(\lambda) = \log\mathbb{E}e^{\lambda X}\) (finite near \(0\)).

Output

The rate function $^* $ (Legendre transform), and from it the answer to the actual question — e.g. BRW speed: \(v^*\) solves \[\Lambda^*(v^*) = \log m,\] balancing \(e^{n\log m}\) particles against \(e^{-n\Lambda^*(v)}\) cost per particle.

What It Converts

  • from: an asymptotic probability question about \(n\)-step sums
  • to: a calculus problem (optimize \(\lambda a - \Lambda(\lambda)\))

Core Idea

The upper bound is one line of Chernoff. The lower bound is the real lesson: tilt the measure (\(d\mathbb{P}_\lambda \propto e^{\lambda X}d\mathbb{P}\)) so the rare event becomes typical, and pay \(e^{-n\Lambda^*(a)}\) in likelihood — see Change of measure. The optimal \(\lambda^*\) is not bookkeeping: it is the tilted world in which the extremal particle lives (the “spine” picture in BRW).

When Should I Think of This?

  • Any probability decaying exponentially in a natural parameter.
  • “Exponentially many independent-ish trials, exponentially small success” — solve entropy = energy.
  • Before trusting any heuristic front-speed claim: check it against \(\Lambda^*(v) = \log m\).

Typical Use Cases

  • Gaussian steps: \(\Lambda^*(a) = a^2/2\) ⇒ binary BRW speed \(v^* = \sqrt{2\log 2}\). (My standard sanity example.)
  • Tail of one DGFF height (Gaussian case of Cramér).
  • Sub-exponential corrections start where this tool stops — handoff to Use second moment method for existence lower bound with barriers.
