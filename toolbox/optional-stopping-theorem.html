Theorem

Theorem: Optional stopping theorem

Statement

If \((M_n)\) is a martingale and \(\tau\) a stopping time, then \(\mathbb{E}[M_\tau] = \mathbb{E}[M_0]\) under any of: (a) \(\tau\) bounded; (b) \(\tau < \infty\) a.s. and \((M_{n \wedge \tau})\) bounded; (c) \((M_{n\wedge\tau})\) uniformly integrable.

Intuition

You cannot beat a fair game with a quitting strategy — provided you can’t hide unbounded losses in the tail.

When to Use

The identity-generating machine: find a harmonic function \(h\) of the chain (so \(h(X_n)\) is a martingale), stop at an exit time, and read off an equation relating hitting probabilities to boundary values.

What It Converts

A martingale (dynamic, path-level object) into a single algebraic identity at a stopping time.

My Standard Applications

  • Gambler’s ruin: \(h(x) = x\) on \(\{0,\dots,n\}\).
  • \(a(X_{n \wedge \tau_0})\) for SRW on \(\mathbb{Z}^2\): the engine of Replace divergent Green function by potential kernel, giving \(\mathbb{P}_x(\tau_{\partial B_n} < \tau_0) = a(x) / (\tfrac{2}{\pi}\log n + \kappa + o(1))\).
  • Wald-type identities for exit times.

Typical Failure Modes

Always check a condition before writing \(\mathbb{E}[M_\tau] = M_0\). Canonical counterexample: SRW on \(\mathbb{Z}\), \(\tau\) = hitting time of \(1\); then \(\mathbb{E}[M_\tau] = 1 \ne 0\) — \(\tau\) is a.s. finite but the stopped martingale is unbounded below. In my 2D applications the saving grace is usually that the domain is finite (condition (b) with boundedness on the box).
