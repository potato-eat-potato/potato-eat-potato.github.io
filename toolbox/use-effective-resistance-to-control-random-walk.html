Tool: Use effective resistance to control random walk

One-Sentence Summary

Reduce a walk question to “compute or bound one number, \(R_{\mathrm{eff}}\)” — then attack that number with the variational vise.

What Problem Does This Solve?

Escape probabilities, recurrence/transience, expected visit counts, commute times — anything in the dictionary of Turn random walk into electrical network.

Input

A reversible network and two “terminals” (vertex ↔︎ vertex, vertex ↔︎ boundary, vertex ↔︎ infinity).

Output

A two-sided estimate on a walk quantity via \[\mathbb{P}_a(\tau_z < \tau_a^+) = \frac{1}{c_a\,R_{\mathrm{eff}}(a,z)}, \qquad G_D(x,x) = c_x\,R_{\mathrm{eff}}(x, D^c).\] The total-conductance factor \(c_a = \sum_{y\sim a} c_{ay}\) is not optional (my RW/EN notes, Remark 6.1); \(c_a = 2d\) for SRW on \(\mathbb{Z}^d\), so escape on \(\mathbb{Z}^2\) is \(1/(4R_{\mathrm{eff}})\).

What It Converts

  • from: walk behavior (paths, return times)
  • to: one scalar with two variational characterizations

Core Idea

\(R_{\mathrm{eff}}\) is squeezed in a vise: - any unit flow \(\theta\): \(R_{\mathrm{eff}} \le \mathcal{E}(\theta)\) — Use Thomson principle by constructing flow; - any test potential \(f\): \(R_{\mathrm{eff}} \ge 1/\mathcal{E}(f)\) — Dirichlet principle; - plus surgery: short for lower bounds, cut for upper bounds — Use Rayleigh monotonicity by shorting or cutting.

So a walk problem becomes: design one clever flow and one clever potential.

When Should I Think of This?

  • The exact Green function/harmonic function is uncomputable but I only need the order of an escape probability.
  • The graph is irregular (holes, perturbations, random environment) — resistance is robust under surgery; exact formulas are not.

Typical Use Cases

  • \(R_{\mathrm{eff}}(0,\partial B_n) \asymp \log n\) in \(\mathbb{Z}^2\) ⇒ escape probability \(\asymp 1/\log n\).
  • Transience of \(\mathbb{Z}^3\) via one finite-energy flow.
  • Perturbed domains: Random walk on box with mesoscopic hole — resistance feels the hole through its capacity.

Where This Appears in My Path

The main technical skill I took from the Zeitouni project; the “so what” slide of my HRUMC talk (resistance growth ⇒ recurrence of \(\mathbb{Z}^2\)).

Current Understanding

Comfortable; I can run both jaws of the vise on radially symmetric examples without notes.

Open Confusions

  • How tight can flow/potential pairs be made in non-symmetric geometries? (Duality gap is zero in principle — the art is explicit constructions.)
