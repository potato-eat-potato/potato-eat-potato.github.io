Object: Green function

One-Sentence Summary

The Green function is the single object where my two reading projects meet: it is simultaneously a random-walk quantity (expected visits), an analytic quantity (inverse Laplacian), and a field quantity (DGFF covariance).

Definition

For transient walks on the whole lattice: \(G(x,y) = \mathbb{E}_x\big[\sum_{k\ge 0} \mathbf{1}\{X_k = y\}\big]\) (finite only for \(d \ge 3\)). In a finite domain, use the Killed Green function \(G_D\). In \(d \le 2\) the whole-space version diverges — see Potential kernel.

The Hub Property

This object connects, and I should always be able to walk this list from memory:

  • Random walk occupation time: expected number of visits.
  • Inverse Laplacian: \(G = L^{-1}\) with \(L = I - P\), i.e. \(LG(\cdot, y) = \delta_y\) — see Use Green function as inverse Laplacian.
  • Dirichlet problem: integral kernel that solves \(-\Delta u = f\).
  • DGFF covariance: \(\mathbb{E}[\varphi_x \varphi_y] = G_D(x,y)\) — see Express DGFF covariance by Green function.
  • Potential kernel: its renormalized recurrent-dimension replacement.
  • Effective resistance: the identity below.

The Identity That Joins My Two Reading Projects

\[G_D(x,x) = c_x\, R_{\mathrm{eff}}(x, D^c),\]

where \(c_x\) is the total conductance at \(x\) (\(c_x = 4\) for SRW on \(\mathbb{Z}^2\)). Proof in one line: the number of visits to \(x\) before exiting is geometric, with success probability the escape probability \([c_xR_{\mathrm{eff}}]^{-1}\). Sanity check on the box \(B_n \subset \mathbb{Z}^2\): \(R_{\mathrm{eff}}(0,\partial B_n) = \frac{\log n}{2\pi} + O(1)\) gives \(G_{B_n}(0,0) = \frac{2}{\pi}\log n + O(1)\). ✓

Zeitouni-side resistance estimates and Bourgade-side field variances are the same numbers.

Important Properties

  • Symmetric: \(G_D(x,y) = G_D(y,x)\) (reversibility).
  • Monotone in the domain: \(D' \subset D \Rightarrow G_{D'} \le G_D\) (fewer visits before exit) — input to Compare domains by monotonicity.
  • \(d \ge 3\): \(G(0,x) \asymp |x|^{2-d}\).
  • Positive semidefinite as a matrix — which is why it can be a covariance.

Why This Object Is Central to My Probability Route

My HRUMC talk built the public half of this story — random walks → harmonic functions → networks → recurrence of \(\mathbb{Z}^2\) — and the Green function is exactly where the sequel begins. Every future direction I care about (extremes of log-correlated fields, cover times, punctured domains) starts by estimating a Green function.

Common Normalizations or Conventions

Vault convention: \(G_D\) counts visits (including time \(0\)) of the walk with \(\Delta = P - I\). Sources using the graph Laplacian differ by a factor \(2d\). Always recompute \(G\)(adjacent) in a new source before trusting its formulas.
