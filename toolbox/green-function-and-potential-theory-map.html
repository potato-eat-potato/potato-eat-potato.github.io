Green Function and Potential Theory Map

The hinge of the vault: everything walk-side and everything field-side passes through here. Public face: Exposition - Why Green Functions Connect Random Walks and DGFF.

The One Object, Six Faces

Green function is simultaneously: 1. expected visits before exit (Killed Green function); 2. the inverse of \(-\Delta\) (Use Green function as inverse Laplacian); 3. the kernel solving Dirichlet/Poisson problems; 4. the covariance of the DGFF (Express DGFF covariance by Green function); 5. resistance in disguise: \(G_D(x,x) = c(x)\,R_{\mathrm{eff}}(x, D^c)\); 6. in \(d=2\), the renormalized limit that is the Potential kernel.

The 2D Storyline (my home dimension)

No Green function on all of \(\mathbb{Z}^2\) → Replace divergent Green function by potential kernel → \(a(x) = \frac{2}{\pi}\log|x| + \kappa + O(|x|^{-2})\) → killed Green functions via \(G_D(x,y) = \mathbb{E}_x[a(X_\tau - y)] - a(x-y)\) → every constant in my escape probabilities and field variances.
