Object: Potential kernel

One-Sentence Summary

\(a(x)\) is what the Green function becomes in recurrent dimension two: you can’t count total visits (infinite), but you can count the visit deficit relative to the origin.

Definition

For SRW on \(\mathbb{Z}^2\): \[a(x) = \sum_{k=0}^{\infty}\Big[\mathbb{P}_0(X_k = 0) - \mathbb{P}_x(X_k = 0)\Big].\]

Important Properties

  • \(a(0) = 0\), \(a \ge 0\), and \(\Delta a = \delta_0\) — so \(a\) is harmonic away from the origin and plays the role of \(-G\).
  • Exact value at neighbors: \(a(e_1) = 1\). One-line proof: \(1 = \Delta a(0) = \frac{1}{4}\sum_{|e|=1} a(e) = a(e_1)\) by symmetry. (This tiny identity computes Effective resistance between adjacent vertices in Z2.)
  • Asymptotics: \(a(x) = \frac{2}{\pi}\log|x| + \kappa + O(|x|^{-2})\), with \(\kappa = \frac{2\gamma + \log 8}{\pi} \approx 1.029\).
  • Uniqueness: \(a\) is the unique nonnegative solution of \(\Delta a = \delta_0\), \(a(0)=0\) with subpolynomial growth — uniqueness needs the growth condition (a Maximum principle-at-infinity issue).

Key Formula

\[G_D(x,y) = \mathbb{E}_x\big[a(X_{\tau_{D^c}} - y)\big] - a(x-y).\] Every killed Green function in \(\mathbb{Z}^2\) is “boundary average of \(a\) minus \(a\).” This is the workhorse behind all my 2D estimates — see Replace divergent Green function by potential kernel.

Typical application via Optional stopping theorem: \(a(X_{n \wedge \tau_0})\) is a martingale, which gives \[\mathbb{P}_x(\tau_{\partial B_n} < \tau_0) = \frac{a(x)}{\frac{2}{\pi}\log n + \kappa + o(1)}.\]

Why This Object Is Central to My Probability Route

I live in \(d = 2\), where the Green function is dead on arrival. The potential kernel is the prosthetic that makes 2D potential theory run; the DGFF variance asymptotics I use in the Bourgade reading come from \(a\)’s log expansion.

Where I Encountered This

Zeitouni project (escape probabilities); Lawler’s books as reference.
