Tool: Turn hitting probability into harmonic function

One-Sentence Summary

Recognize a hitting/exit probability as a harmonic function with known boundary values, and from then on use analysis instead of path-counting.

What Problem Does This Solve?

Any “which target does the walk reach first?” question — gambler’s ruin, escape probabilities, exit distributions — where direct path analysis is hopeless.

Input

A Markov chain, two disjoint target sets \(A\) and \(B\), the function \(h(x) = \mathbb{P}_x(\tau_A < \tau_B)\).

Output

The fact that \(h\) is the unique solution of the discrete Dirichlet problem: \(\Delta h = 0\) off \(A \cup B\), \(h|_A = 1\), \(h|_B = 0\).

What It Converts

  • from: a probabilistic quantity defined over infinitely many paths
  • to: a boundary value problem for the Discrete Laplacian

Core Idea

Condition on the first step: \(h(x) = \frac{1}{2d}\sum_{y \sim x} h(y)\) — that is harmonicity. Uniqueness comes from the Maximum principle. From here, every harmonic-function technique (explicit solutions, comparison, monotonicity, energy) applies to a probability question. The reverse direction is Convert boundary value problem into probabilistic representation.

When Should I Think of This?

  • The unknown is a probability of one event preceding another.
  • Someone hands me a harmonic function and I want to know what it “counts.”
  • I need uniqueness: two candidate answers agreeing on the boundary must agree everywhere.

Typical Use Cases

  • Gambler’s ruin on \(\{0, \dots, n\}\): \(h\) linear.
  • Escape probability via potential kernel: \(a(X_n)\) harmonic off \(0\) + Optional stopping theorem ⇒ Escape probability of SRW on Z2.
  • Voltage = hitting probability: the first plank of Turn random walk into electrical network.

Where This Appears in My Path

This was the opening move of both my talks — the first slide of the HRUMC talk and the first half-hour of the Courant blackboard talk were exactly this conversion. With Zeitouni it was the entry point into Doyle–Snell.

Current Understanding
