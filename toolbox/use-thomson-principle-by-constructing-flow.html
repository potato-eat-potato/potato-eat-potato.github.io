Tool: Use Thomson principle by constructing flow

One-Sentence Summary

To bound a resistance from above, don’t solve the network — build any unit flow and pay its energy.

What Problem Does This Solve?

Upper bounds on \(R_{\mathrm{eff}}\), hence: lower bounds on escape probabilities, proofs of transience, upper bounds on \(G_D(x,x)\).

Input

A network and terminals; an ansatz for how current “should” spread (usually stolen from the continuum).

Output

\(R_{\mathrm{eff}} \le \mathcal{E}(\theta) = \sum_e \theta(e)^2 r_e\) for the constructed unit flow \(\theta\).

Construction Recipe

  1. Guess the continuum picture (radial spreading, flow along a spanning tree, uniform spreading over shells).
  2. Discretize: define \(\theta(e)\) explicitly; verify the node law (this is where constructions silently die).
  3. Sum the energy shell by shell.

Worked reflex example — transience of \(\mathbb{Z}^3\): spread unit flow so each edge at distance \(k\) carries \(\asymp k^{-2}\); shells have \(\asymp k^2\) edges; \(\mathcal{E} \asymp \sum_k k^2 \cdot k^{-4} < \infty\). Done.

What It Converts

  • from: a global optimization (true current)
  • to: a single explicit construction + a sum

When Should I Think of This?

  • I need “the walk escapes” / “resistance is finite/small.”
  • I can picture how water would drain through the graph — that picture is the flow.
  • Dual jaw: if I instead need resistance large, switch to test potentials (Dirichlet principle) or shorting (Use Rayleigh monotonicity by shorting or cutting).

Typical Use Cases

  • Transience of \(\mathbb{Z}^d\), \(d \ge 3\).
  • Trees: flow splitting along rays computes resistance to infinity exactly.
  • The superposition flow in Effective resistance between adjacent vertices in Z2.
