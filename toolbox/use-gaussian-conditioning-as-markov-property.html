Tool: Use Gaussian conditioning as Markov property

One-Sentence Summary

Conditioning the DGFF on some vertices splits it into (deterministic harmonic extension) + (independent zero-boundary DGFF on what remains) — the field’s substitute for the strong Markov property.

What Problem Does This Solve?

Computing conditional laws of the field given boundary data, given values on a subset, or given the coarse field — the basic move for decompositions, comparisons, and the punctured box.

Input

Jointly Gaussian structure + the Gaussian conditioning formula: conditional mean is a linear projection; conditional covariance is deterministic (independent of the observed values!).

Output

Domain Markov / Gibbs–Markov decomposition. For \(U \subset D\): \[\varphi_D \stackrel{d}{=} \underbrace{\varphi_U}_{\text{indep. zero-bdry DGFF on } U} + \underbrace{\mathsf{H}[\varphi_D|_{D \setminus U}]}_{\text{harmonic extension into } U}.\] The conditional covariance inside \(U\) is \(G_U\) — conditioning shrinks the domain of the Green function.

What It Converts

  • from: conditional probability (measure theory)
  • to: linear algebra (projections) + potential theory (harmonic extension, killed Green functions)

Core Idea

For Gaussians, conditioning = orthogonal projection in the covariance inner product. The projection onto “values on \(D\setminus U\)” is computed by harmonic extension because \(G_D\) inverts the Laplacian. Two beautiful special cases:

  1. One point (\(U = D \setminus \{v\}\), condition \(\varphi_v = 0\)): the conditioned field is the DGFF on the punctured domain, and the covariance update is the puncture identity in Killed Green function. So DGFF on punctured box = “Gaussian conditioning, taken seriously.”
  2. Dyadic blocks: iterate over scales ⇒ the hierarchical/BRW structure — Use multiscale decomposition.

When Should I Think of This?

  • Any “given the field here, what happens there?” question.
  • Whenever I want independence: carve the domain so the pieces decouple after conditioning.
  • When comparing fields on different domains (couple them by conditioning the bigger one).

Where This Appears in My Path
