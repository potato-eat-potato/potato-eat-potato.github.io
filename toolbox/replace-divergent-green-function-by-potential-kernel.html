Tool: Replace divergent Green function by potential kernel

One-Sentence Summary

In recurrent dimension two, run every Green-function argument with \(a(x)\) in place of \(G\): same harmonicity, same martingales, finite answers.

What Problem Does This Solve?

All of 2D potential theory, where \(G(x,y) = \infty\) kills the standard transient-case formulas.

Input

The Potential kernel \(a\) with its three working properties: \(\Delta a = \delta_0\); \(a(e_1) = 1\); \(a(x) = \frac{2}{\pi}\log|x| + \kappa + O(|x|^{-2})\).

Output

Finite-volume Green functions and hitting probabilities by pure bookkeeping: \[G_D(x,y) = \mathbb{E}_x\big[a(X_{\tau} - y)\big] - a(x - y).\]

What It Converts

  • from: divergent occupation-time sums
  • to: boundary averages of one explicit function

Core Idea

\(a(X_n - y)\) is a martingale while the walk avoids \(y\) (harmonicity off the origin), so Optional stopping theorem at exit/hitting times produces identities. The asymptotic expansion then converts identities into estimates with explicit constants. Standard dividend: \[\mathbb{P}_x(\tau_{\partial B_n} < \tau_0) = \frac{a(x)}{\tfrac{2}{\pi}\log n + \kappa + o(1)},\] which at \(x = e_1\) (where \(a = 1\)) gives the sharp escape probability \(\sim \pi/(2\log n)\).

When Should I Think of This?

  • Any \(\mathbb{Z}^2\) quantity that would be a one-liner with a Green function in \(d \ge 3\).
  • When constants matter: \(a\)’s expansion carries the exact \(\frac{2}{\pi}\) and \(\kappa\) that resistance shorting arguments lose.
  • When a quantity is only defined “relative to the origin” — \(a\) is precisely the relative object.

Typical Use Cases

  • Escape probability of SRW on Z2 — sharp version.
  • Effective resistance between adjacent vertices in Z2 via \(a(e_1) = 1\).
  • DGFF variance asymptotics: \(G_{B_n}(x,x)\) via the boundary-average formula — this is where the \(\frac{2}{\pi}\log N\) in Express DGFF covariance by Green function comes from.
