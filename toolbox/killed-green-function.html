Object: Killed Green function

One-Sentence Summary

\(G_D\) — the Green function of the walk killed on exiting \(D\) — is the finite-domain workhorse: every DGFF computation I do starts here.

Definition

For \(D \subsetneq \mathbb{Z}^d\) and \(\tau = \tau_{D^c}\) the exit time, \[G_D(x,y) = \mathbb{E}_x\Big[\sum_{k=0}^{\tau - 1} \mathbf{1}\{X_k = y\}\Big].\] Equivalently: with the Dirichlet/killed Laplacian \(L = I - P\) on \(D\), \(\;L G_D(\cdot, y) = \delta_y\) and \(G_D(\cdot, y) = 0\) outside \(D\) — so \(G_D = L^{-1}\) (my DGFF notes, Prop. 5).

Intuition

Visits before you fall off the edge of the domain. Killing is what makes the inverse exist — on all of \(\mathbb{Z}^2\) there is no inverse (Potential kernel is the patch).

Key Formulas I Actually Use

  • Puncture identity (= Gaussian conditioning at one point): \[G_{D\setminus\{v\}}(x,y) = G_D(x,y) - \frac{G_D(x,v)\,G_D(v,y)}{G_D(v,v)}.\] This is the engine of DGFF on punctured box.
  • Potential kernel representation in \(\mathbb{Z}^2\): \[G_D(x,y) = \mathbb{E}_x\big[a(X_\tau - y)\big] - a(x-y).\] See Replace divergent Green function by potential kernel.
  • Box asymptotics (\(d=2\), \(x\) deep inside \(B_n\)): \(G_{B_n}(x,x) = \frac{2}{\pi}\log n + O(1)\); off-diagonal \(G_{B_n}(x,y) \approx \frac{2}{\pi}\log\frac{n}{|x-y|}\) — the source of log-correlations of the DGFF.

Important Properties

  • Symmetric, positive semidefinite, monotone increasing in \(D\).
  • Zero boundary values — the analytic reason the DGFF has the domain Markov property.

Why This Object Is Central to My Probability Route

The Bourgade reading direction is essentially: understand fields whose covariance is \(G_D\). Any control I want on the field (variance, correlation, effect of removing a set) must first be proven about \(G_D\).

Tools That Use This Object

  • Express DGFF covariance by Green function
  • Use Gaussian conditioning as Markov property
  • Regularize infinite graph by finite box

Where I Encountered This

Zeitouni’s Gaussian field notes (as covariance), and concretely while thinking about the punctured box with the Bourgade reading.
