Tool: Regularize infinite graph by finite box

One-Sentence Summary

When an infinite-volume object diverges or doesn’t exist, define everything in a box \(B_n\), prove estimates uniform in \(n\), and only then discuss \(n \to \infty\).

What Problem Does This Solve?

Divergences that are artifacts of infinite volume: no Green function on \(\mathbb{Z}^2\), no infinite-volume 2D DGFF, “resistance to infinity” needing definition.

Input

The infinite-lattice question; an exhaustion \(B_n \uparrow \mathbb{Z}^d\).

Output

A family of well-posed finite problems plus a limit statement. Three standard outcomes:

  1. Monotone limit exists: \(R_{\mathrm{eff}}(0, \partial B_n) \uparrow R_{\mathrm{eff}}(0,\infty)\) — recurrence iff \(= \infty\). The limit is the definition.
  2. Limit exists after recentering/differencing: \(G_{B_n}(0,0) - G_{B_n}(x,0) \to a(x)\) — the Potential kernel is born as a renormalized finite-volume limit. Gradients of the 2D DGFF converge even though the field itself doesn’t.
  3. No limit, and that’s the theorem: the 2D DGFF has \(\mathrm{Var}(\varphi_0) = \frac{2}{\pi}\log n \to \infty\); all statements must stay finite-volume with explicit \(n\)-dependence. (In \(d \ge 3\), the infinite-volume field exists since \(G(0,0) < \infty\).)

What It Converts

  • from: ill-posed infinite-volume questions
  • to: finite linear algebra + a limit argument with explicit error bookkeeping

Core Idea

Killing at \(\partial B_n\) makes \(-\Delta\) invertible. The discipline that makes the tool research-grade: always record how constants depend on \(n\) — in \(d=2\) the \(n\)-dependence (the \(\log\)) is usually the answer, not the error.

When Should I Think of This?

  • Anything diverges → put it in a box before debugging.
  • A quantity is “defined up to additive constant” → it’s a finite-volume difference in disguise.
  • Comparing two infinite-volume heuristics → compare them in the same box first.

Where This Appears in My Path

Both reading projects forced it: recurrence-as-resistance-limit (Zeitouni side), the nonexistence of the 2D infinite-volume DGFF (Bourgade side). It’s also a pattern — proof-skeleton version at Finite volume approximation.
