Theorem: Maximum principle

Statement

A function harmonic on a finite connected \(D\) attains its maximum (and minimum) on the boundary \(\partial D\); if attained inside, the function is constant. Consequence: uniqueness for the discrete Dirichlet problem — harmonic functions are determined by boundary values.

Intuition

Mean-value property forbids interior peaks: a strict interior max would have to exceed the average of its neighbors. Probabilistically: \(h(X_n)\) is a martingale, and a fair game can’t be sure to go down.

When to Use

  • Identification: two candidate formulas, both harmonic, same boundary values ⇒ equal. This is the glue in Turn random walk into electrical network — voltage equals hitting probability because both solve the same Dirichlet problem.
  • Comparison: \(h_1 \le h_2\) on \(\partial D\) and both harmonic ⇒ \(h_1 \le h_2\) everywhere (Boundary-to-bulk comparison).
  • Subharmonic versions for one-sided bounds.

What It Converts

A pointwise interior claim into a boundary check — infinitely cheaper.

Proof Sketch

Walk from an interior max along neighbors realizing the average; connectivity drags the max to the boundary. (Or: optional stopping on \(h(X_{\tau})\).)

Typical Failure Modes

  • Unbounded domains: uniqueness fails without growth conditions (\(h(x) = x_1\) is harmonic on \(\mathbb{Z}^2\), zero nowhere relevant). This is exactly why the Potential kernel’s uniqueness statement carries a sublinearity hypothesis.
  • Disconnected domains: apply per component.
  • Operators with zeroth-order terms (killing): still fine; with wrong signs: not fine. Check before generalizing.

Related Tools

  • Turn hitting probability into harmonic function
  • Use Green function as inverse Laplacian (uniqueness of the Poisson solve)
