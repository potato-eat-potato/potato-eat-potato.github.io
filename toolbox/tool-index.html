Tool Index

Every tool in 02 Tools, grouped by the kind of move, with a one-line “reach for this when.” Update on every new tool.

A. Walk ↔︎ Analysis ↔︎ Network conversions

  • Turn hitting probability into harmonic function — when the unknown is “which target first?”
  • Turn random walk into electrical network — when I want surgery and energy bounds on a reversible walk.
  • Use effective resistance to control random walk — when escape/recurrence/visits should become one number.
  • Convert boundary value problem into probabilistic representation — (to write) the reverse direction: solve PDEs by running walks.

B. Variational / energy moves

  • Use Thomson principle by constructing flow — when I need resistance ≤ something: build a flow.
  • Use Rayleigh monotonicity by shorting or cutting — when surgery to a solvable circuit settles the order. Short = lower, cut = upper.
  • Use energy minimization to prove resistance bounds — (to write; meanwhile see the pattern Energy minimization argument)

C. Green function / potential theory moves

  • Use Green function as inverse Laplacian — when an equation reads “\(-\Delta u = f\)” or a sum reads “reward until exit.”
  • Replace divergent Green function by potential kernel — when a transient-case formula dies in \(d=2\).
  • Regularize infinite graph by finite box — when anything diverges: box first, limit later.
  • Compare domains by monotonicity — when the domain is awkward: sandwich it.

D. Gaussian field moves

  • Express DGFF covariance by Green function — first question for any DGFF statement: what does it say about \(G_D\)?
  • Use Gaussian conditioning as Markov property — when conditioning the field: harmonic mean + killed covariance.
  • Use multiscale decomposition — when the question is about extremes/fine structure of a log-correlated field.

E. Moment methods and extremes

  • Use first moment method for maximum upper bound — always run first on any max question; free calibration.
  • Use second moment method for existence lower bound — to match the first moment; truncate when correlated.

F. Large deviations

  • Use Cramer transform for large deviation speed — exponential decay rates and front speeds: \(\Lambda \to \Lambda^* \to\) solve.
